# 元素水平居中和垂直居中的实现方案有哪些？

# 水平居中

# 行内元素水平居中

利用 text-align: center 可以实现对块级元素内部（父元素）的行内元素（子元素）进行水平居中。如果子元素是inline、inline-block、inline-table 和 inline-flex元素，均可以使用该方法。如果子元素有多个，需要保证每个元素都是行内元素，这样才能排列在一行。但是，子元素里的文字也会水平居中，可以为子元素添加text-align:left 让文字居左对齐。

// 父元素添加样式
.parent {
    text-align:center;
}

# 块级元素水平居中

# 方法一、将块级元素转为行内块级元素，再给父元素应用 text-align: center

// 父元素添加样式
.child {
    display: inline-block;
  }
.parent {
    text-align:center;
}

# 方法二、为块级元素应用 margin:0 auto 进行水平居中。需要注意的是，该元素必须有宽度。

// 当前块级元素添加样式
.current {
    width：300px;
    margin: 0 auto;
  }

# 方法三、如果块级元素没有宽度，使用 display: table、margin: 0 auto 进行水平居中。display:table 在表现上类似 block 元素，但是宽度为内容宽。

// 当前块级元素添加样式
.current {
    display: table;
    margin: 0 auto;
  }

# 方法四、使用 absolute + transform，先将父元素设置为相对定位，再将子元素设置为绝对定位，left 为父容器的一半，最后通过 transform 向左移动子元素的一半宽度以达到水平居中。注意，子元素不需要指定宽度。

.parent {
    position:relative;
  }
.child {
    position: absolute;
    left: 50%;
    transform: translateX(-50%);
  }

# 方法五、使用 absolute + margin，先将父元素设置为相对定位，再将子元素设置为绝对定位，left 为父容器的一半，指定负margin为元素的一半宽度以达到水平居中。注意，子元素需要指定宽度。

.parent {
    position:relative;
  }
.child {
    position: absolute;
    width: 200px;
    left: 50%;
    margin: -100px;
  }

# 方法六、使用 absolute + margin: 0 auto，先将父元素设置为相对定位，再将子元素设置为绝对定位，left 为 0，right 为 0，指定margin: 0 auto以达到水平居中。注意，子元素需要指定宽度。

.parent {
     position:relative;
   }

.child{
     position: absolute;
     width: 200px;
     height:100px;
     background: yellow;
     margin: 0 auto; 
     left: 0;
     right: 0;
   }

# 方法七、使用 flex 布局

.parent {
    display: flex;
    justify-content: center;
  }

# 多个块级元素水平居中

# 方法一、使用 flex 布局，会将多个块级元素排列到一行

.parent {
    display: flex;
    justify-content: center;
  }

# 方法二、将要水平排列的块状元素设为 display:inline-block，然后在父级元素上设置 text-align:center。

.parent {
   text-align: center;
}
.child {
   display: inline-block;
}

# 浮动元素水平居中

# 方法一、使用 flex 布局（浮动元素是否有固定宽度都可以实现水平居中）

.parent {
    display:flex;
    justify-content:center;
}
.child {
    float: left;
}

# 方法二、浮动元素如果有固定宽度，使用 relative + 负margin。

.child {
    float: left;
    width:200px;
    position:relative;
    left:50%;
    margin-left:-250px;
}

# 方法三、浮动元素没有固定宽度，无法使用负margin，需要父子容器都用相对定位。

父子元素都浮动后，会堆叠在页面左边，然后父元素通过left向右移动页面的一般，子元素用right向左移动父元素的一半，实现了水平居中。

.parent {
    float:left;
    position:relative;
    left:50%;
}
child {
    float:left;
    position:relative;
    right:50%;
}

# 垂直居中

# 单个行内元素垂直居中

.parent {
    height: 120px;
    line-height: 120px;
}

# 多个行内元素垂直排列并居中

# 方法一、利用 table 和 table-cell 布局

.parent {
     display: table;
     height: 140px;
     border: 2px dashed #f69c55;
}
.child {
     display: table-cell;
     vertical-align: middle;
}

# 方法二、利用 flex 布局

.parent {
    height: 140px;
    display: flex;
    flex-direction: column;
    justify-content: center;
}

# 多个块级元素垂直居中

# 方法一、使用absolute+负margin(已知高度宽度)

.parent {
    position: relative;
}
.child {
    width: 100px;
    position: absolute;  
    top: 50%;
    height: 100px;
    margin-top: -50px;
}

# 方法二、使用absolute+transform(高度和宽度未知)

.parent {
    position: relative;
}
.child {
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
}

# 方法二、使用flex+align-items

.parent {
    display:flex;
    align-items:center;
}

# 方法二、使用table-cell+vertical-align

.parent {
    display: table-cell;
    vertical-align: middle;
}

# 水平和垂直均居中

对上述水平居中和垂直居中方法进行综合，均可以实现元素的水平垂直居中。

# 单行行内元素（可以有多个行内元素）

# 方法一、使用 text-align、line-height

.parent {
    width:300px;
    height:400px;
    text-align: center;
    line-height: 400px;
}

# 方法二、使用 flex 布局

.parent {
    display:flex;
    justify-content: center;
    align-items:center;
}

# 多个块级元素水平、垂直居中

# 方法一、使用 absolute + 负margin (已知高度宽度)

使用margin处理居中必须知道元素的宽高，因为margin如果使用百分比，百分比拾亿父元素的宽度为基准的，即使是设置垂直方向的margin-top，使用百分比，也是以父元素宽度为基准，而不是高度。所以为margin应用百分比是推荐的方法。所以只能设置负margin为自己宽度的一半，而不能使用50%这样的设置。选用绝对定位， 因为它对整个布局的影响太过强烈。

.parent {
    position: relative;
}
.child {
    width:300px;
    height: 200px;
    position: absolute;
    left: 50%;
    top: 50%;
    margin-left: -150px;
    margin-top: -100px;
}

# 方法二、使用 absolute + calc函数计算代替 负 margin (已知高度宽度)

.parent {
    position: relative;
}
.child {
    width:300px;
    height: 200px;
    position: absolute;
    top: calc(50% - 100px);
    left: calc(50% - 150px);
}

# 方法二、使用 absolute + margin: auto，该方法不需要知道子元素的固定宽高。

.parent {
    position: relative;
}
.child {
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    margin: auto;
}

# 方法三、使用 absolute + transform（不需要知道子元素的宽高）

transform 与 margin 不同，transform 是以元素自身宽高记性计算的，所以，无需知道元素的宽高，使用百分比就可以计算出来。选用绝对定位， 因为它对整个布局的影响太过强烈。

.parent {
    position: relative;
}
.child {
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%,-50%);
}

# 方法四、使用 flex 布局（父元素需要有高度）。

justify-content、align-items同样可以实现文本的居中。

.parent {
    height: 300px;
    display: flex;
    justify-content: center;
    align-items: center;
}

# 方法五、flex/grid 与 margin:auto（父元素需要有高度）

margin: auto 不仅在水平方向上将元素居中， 垂直方向上也是如此。

.parent {
    height: 300px;
    display: grid;
}
.child {
    margin: auto;
}

# 方法五、使用视口单位vh margin：auto实现居中

50vh代替，margin-top是视口高度的一半，auto代表水平居中。然后使用 transform 让元素沿Y轴向上移动自身高度的一半。该方法的局限性在于，只适用于元素在视口中居中。

.current {
    margin: 50vh auto 0;
    transform:translateY(-50%)
}

# 方法五、align-self: center 实现垂直居中

align-items应用在父元素上，align-self应用在子元素上，如果父元素统一设置了align-items，子元素上设置align-self，可以覆盖该元素应用的align-items的对齐方式。

.parent {
    height: 300px;
    display: flex;
    justify-content: center;
}
.child {
    align-self: center
}

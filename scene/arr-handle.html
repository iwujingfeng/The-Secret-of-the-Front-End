# 数组处理

# 获取元素在数组中的索引

const arr = ["jack", "mike", "rose"];
const index = arr.indexOf("rose");
console.log("indexOf", index); // 2

const mikeIndex = arr.findIndex(item => item === "mike");
console.log("findIndex", mikeIndex); // 1

案例解析

最简单的方式就是使用 indexOf()方法获取元素在数组中的索引值。不过，findIndex()提供了另一种思路，通过查找遍历出的元素中和需要获取索引的元素是否相同，实现这个需求，返回第一个符合条件的元素的索引。

# 数组的增删改查

# 增加元素

const arr = ["mike"];
arr.push("rose");
arr.unshift("jack");
console.log("add", arr); // [ 'jack', 'mike', 'rose' ]

// 从某个索引位置开始增加元素
arr.splice(1, 0, "john");
console.log("add", arr); // [ 'jack', 'john', 'mike', 'rose' ]
// splice实现数组头部增加元素
arr.splice(0, 0, "anne");
console.log("add", arr); // [ 'anne', 'jack', 'john', 'mike', 'rose' ]
// splice实现数组尾部增加元素
arr.splice(arr.length, 0, "emmy");
console.log("add", arr); // [ 'anne', 'jack', 'john', 'mike', 'rose', 'emmy' ]

arr[arr.length] = "martin";
console.log("add", arr); // [ 'anne', 'jack', 'john', 'mike', 'rose', 'emmy', 'martin' ]

案例解析

在数组头部增加元素可以使用 unshift()方法，数组尾部增加元素可以使用 push()。如果要在数组的任意索引位置增加元素，需要使用 splice()，该方法既可以实现数组删除元素，也可以实现数组增加元素。如果 splice()传递三个参数，第一个参数是要删除元素的起始索引位置，第二个参数是删除的元素的数量，第三个参数就是要增加的元素。第一个 splice()方法，执行的是在从索引 1 的位置，删除 0 个元素，然后在该索引位置增加元素"john"，其他元素后移。最关键的操作，是删除 0 个元素这一步。第二个 splice()方法实现的是在数组的头部增加元素，在索引 0 的位置删除 0 项，索引 0 的位置增加"anne"，其他元素后移。第三个 splice()实现的是数组尾部增加元素，其实索引位置必须使用数组的长度 arr.length，因为 length 的值是大于数组的最大索引值，所以会实现在数组尾部增加元素的效果。最后，是直接通过数组索引为数组增加元素，但只能实现在数组尾部增加元素，如果是其他索引，将会导致替换当前位置元素的后果。

说明

观察代码中 push()方法的演示可以发现，unshift()和 push()都是有返回值的，返回值就是增加元素后的数组的长度。

# 删除元素

const arr = ["anne", "jack", "john", "mike", "rose", "emmy", "martin"];
const del = arr.shift();
arr.pop();
console.log("del", arr); // [ 'jack', 'john', 'mike', 'rose', 'emmy' ]
console.log("del", del); // anne
// 从数组中删除指定索引位置的元素
const delEl = arr.splice(1, 1);
console.log("del-index", arr, delEl); // [ 'jack', 'mike', 'rose', 'emmy' ] [ 'john' ]
console.log("del-index", arr.length); // 4
// splice删除数组头部元素
arr.splice(0, 1);
console.log("del", arr); // [ 'mike', 'rose', 'emmy' ]
// splice删除数组尾部元素
arr.splice(arr.length - 1, 1);
console.log("del", arr); // [ 'mike', 'rose' ]

delete arr[0];
console.log("delete", arr); // [ <1 empty item>, 'rose' ]
console.log("delete", arr.length); // 2

// 从数组中删除指定元素
const arr1 = ["jack", "mike", "rose"];
const roseIndex = arr1.indexOf("rose");
arr1.splice(roseIndex, 1);
console.log("del-el", arr1); // [ 'jack', 'mike' ]

案例解析

删除数组头部元素可以使用 shift()方法，删除数组尾部元素可以使用 pop()。从数组中删除指定索引位置的元素，可以使用 splice()方法，第一个参数是删除元素的起始索引位置，第二个参数是删除的数量，会从起始位置开始（包括起始位置元素），向后删除指定数量的元素。通过打印结果可以发现，splice()会改变原始数组，并且返回新的数组的 length。如果只需要删除元素后的原始数组，不需要将 splice()操作的返回值赋值给变量，返回值其实是具体删除的元素，是一个数组。splice()也可以删除数组头部元素和尾部元素，删除尾部元素，尾部元素的索引值是 arr.length - 1。

通过 delete 运算符也可以删除指定索引位置的元素，但删除的索引位置会保留一个空单元，数组的 length 也不会因为删除操作而改变。不推荐delete 运算符，心中有这个概念即可。

数组中删除指定元素，需要先通过 indexOf()获取该元素在数组中的索引。然后执行 splice()操作。在实际开始中，这类场景很常见（不考虑直接请求接口删除），譬如，我们有一个要提交后台的列表，提交前，用户可以增加删除元素，点击某一列表项的删除按钮，通过点击事件获取点击项在数组中的索引，执行 splice()进行删除。

说明

观察代码中 shift()方法的演示可以发现，shift()和 pop()都是有返回值的，返回值就是删除的那一项元素。

# 修改元素

const arr = ["jack", "mike", "rose"];
// 通过索引修改
arr[1] = "john";
console.log("modify", arr); // [ 'jack', 'john', 'rose' ]
// splice修改
arr.splice(2, 1, "anne");
console.log("modify", arr); // [ 'jack', 'john', 'anne' ]
// 把某个元素替换为新的元素
const index = arr.indexOf("john");
arr.splice(index, 1, "emmy");
console.log("modify", arr); // [ 'jack', 'emmy', 'anne' ]
// forEach等遍历修改
const arr1 = ["jack", "john", "mike", "rose", "martin"];
arr1.forEach((item, index) => {
  if (item.indexOf("o") !== -1) {
    arr1[index] = "martin";
  }
});
console.log("modify", arr1); // [ 'jack', 'martin', 'mike', 'martin', 'martin' ]

案例解析

在明确知道修改哪一项元素的情况下，可以通过索引直接对元素进行修改，也可以通过 splice()进行修改，splice()执行的其实是在索引位置，删除一项，增加一项。如果想修改某一个特定的元素，可以先用 indexOf()获取该元素在数组中的索引，然后在执行 splice()进行修改。

可以使用数组的相关方法对数组进行修改，譬如 map()、filter()等过滤方法以及 slice()方法等。具体可以参考应用场景-数组过滤一章。这里以 forEach() 为例，说明修改数组的思路，通过遍历，可以对符合条件的元素进行修改，最终原始数组会进行更新元素。但需要注意的，forEach()遍历后，不能直接对元素进行修改，需要使用索引的方式修改。具体细节差异，可以参考应用场景-数组遍历一章。

# 查找元素

const arr = ["jack", "john", "mike", "rose", "martin"];
console.log(arr[3]); // rose
console.log(arr[arr.length - 1]); // martin 获取数组最后一个值
function findEl(arr) {
  return arr.find(item => {
    return item.includes("o");
  });
}
console.log("find", findEl(arr)); // john
function filterEl(arr) {
  return arr.filter(item => {
    return item.includes("o");
  });
}
console.log("filter", filterEl(arr)); // [ 'john', 'rose' ]

// 解构赋值查找数组元素
const [a, b, ...c] = arr; // 获取前面几个元素
console.log("...", a, b, c); // jack john [ 'mike', 'rose', 'martin' ]

案例解析

获取指定索引位置的元素，可以直接使用索引获取，通过 arr.length - 1 可以后去最后一个元素。这里以 find()、filter()方法举例，实现按照条件查找元素的思路。这两个方法都需要 return 一个查找的条件，最终返回查找的结果，find()返回符合条件的第一个元素，filter()会返回包含全部符合条件的元素的新数组。需要注意的是，这里演示的只是极少数的方法，众多数组的方法都可以实际元素的查找，包括过滤的相关方法、数组截取方法等等，具体可以参考应用场景-数组过滤。

使用 ES6 提供的解构赋值，也可以查找出数组的元素，这里例子演示了查找出数组前两项元素（赋值给a、b变量）的方法，后面的元素会赋值到 c 这个变量上，元素被存储在数组。

# 判断数组中是否包含指定的元素

const arr = ["jack", "mike", "rose"];
console.log("indexOf", arr.indexOf("jack") !== -1); // true
console.log("includes", arr.includes("mike")); // true
const hasEl = arr.find(item => {
  console.log(item);
  return item === "rose";
});
console.log("find", hasEl === "rose"); // true

const hasElIndex = arr.findIndex(item => {
  return item === "rose";
});
console.log("findIndex", hasElIndex !== -1); // true

案例解析

和字符串一样，判断数组中是否包含某个元素，最常用的方法就是indexOf() 和 includes() 方法，在 ES6 之前，通过indexOf()判断某个元素的索引是否等于-1，可以判断该元素是否存在，而 ES6 提供的includes()是更加语义化的方法，推荐使用。通过find()查找出符合条件的元素，判断该元素是否等于查找的元素，也可以实现这个需求，但略显繁琐，相比之下，findIndex()通过判断元素的索引是否等于-1 似乎更加优雅一些。

说明

数组和字符串都有 indexOf() 和 includes() 方法

# 判断数组是否为空

const arr = [];
console.log(arr.length === 0); // true
console.log(JSON.stringify(arr) === "[]"); // true
console.log(+arr === 0); // true

案例解析

判断数组是否是空数组，最常用的方法，就是通过数组的 length 属性判断，空数组的 length 等于 0。通过将数组转换为 JSON 字符串，也可以判断，数组等于字符串"[]"，说明是空数组。第三种方法，不太常用，知道书写形式即可，作为了解。

# 获取一个数组中的最大值和最小值

var arr = [1, 3, 2, 5];
arr.sort((a, b) => a - b);
console.log("max", arr[arr.length - 1]); // 5
console.log("min", arr[0]); // 1

// apply
console.log("max", Math.max.apply(null, arr)); // 5
console.log("min", Math.min.apply(null, arr)); // 1

// 扩展运算符
console.log("max", Math.max(...arr)); // 5
console.log("min", Math.min(...arr)); // 1

案例解析

第一种方法是使用 sort()对数组中的数字进行升序排序，然后通过索引获取最大值和最小值。第二种方法是使用了 apply()方法，关于 apply()方法，可以参考核心技术-apply、call、bind 一章。第三种方法是使用 ES6 提供的扩展运算符，将数组扩展为 Math.max()和 Math.min()可以接受的参数列表形式，获取最大值和最小值。

# 动态移动数组中的元素

const arr = ["jack", "john", "mike", "rose", "martin"];
const currentIndex = arr.indexOf("mike");
arr.splice(0, 0, arr[currentIndex]); // 0位置 删除0项 添加移动的项
arr.splice(currentIndex + 1, 1); // 数组length+1了 删除移动项目时 index要加1
console.log("move", arr); // [ 'mike', 'jack', 'john', 'rose', 'martin' ]

案例解析

有时候，我们需要对数组某一个元素执行操作后，将其移动到其他位置，譬如移动到数组头部或者尾部。举例来说，有一个聊天列表，收到新消息或者发送新消息后的那一个聊天列表项，需要移动到整个列表的顶部。

先获得要移动的数组项的索引（实际开发中，这个索引需要通过点击事件等获取），通过 splice()方法在索引 0 的位置删除 0 项，把要移动的那一项添加进去，其他元素后移。接着删除移动的那一项，因为数组长度增加，移动的元素的索引也应该 +1，然后执行 splice()即可删除原来位置的数组项。

说明

数组的很多方法属于变异方法，会改变原始数组，方法有：push()、pop()、unshift()、shift()、splice()、sort()、reverse()。非变异方法不会改变原始数组，只会返回新的数组或新的值，这些方法有 slice()、filter() 等过滤方法、concat() 方法等等，非变异方法需要将返回值赋值给一个变量保存，或者在封装的函数内部进行 return，如果不想这样，只想改变原始数组，可以将返回值直接赋值给原始数组（变量）。
